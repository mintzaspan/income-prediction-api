# Predict income based on census data
API for model predicting whether a person makes over 50K a year based on census [data](https://archive.ics.uci.edu/dataset/20/census+income).

## How to use
```python
# clone project
git clone https://github.com/mintzaspan/income-prediction-api.git

# move to project dir
cd income-prediction-api/

# create project environment from conda.yml
conda env create -f conda.yml

# activate environment
conda activate income-prediction-api

# install pre-commit hooks
pre-commit install
```
